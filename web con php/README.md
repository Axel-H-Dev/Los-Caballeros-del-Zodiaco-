# Sitio web - Los caballeros del zodiaco

Sitio web hecho con HTML y CSS sobre la serie **Los Caballeros del Zodiaco**.

## Páginas

- **Inicio** (`index.html`) — Portada con video introductorio y descripción de la serie.
- **Galería** (`galeria.html`) — Imágenes y contenido visual.
- **Nosotros** (`nosotros.html`) — Información sobre el sitio.
- **Contáctanos** (`contactanos.html`) — Datos de contacto.

## Tecnologías usadas

- HTML5
- CSS3

## Estructura del proyecto

```
web con php/
├── index.html
├── galeria.html
├── nosotros.html
├── contactanos.html
├── style.css
└── img/
    ├── fondo2.jpg
    ├── dorados.jpg
    └── video intro.mp4
```
